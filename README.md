# Magnum Kitchen Prototype

Short playable Unity prototype for **Magnum** — a small mouse knight exploring a human-scale kitchen.

## Requirements
- Unity: 6000.3.8f1
- Platform tested: Windows

## How to Run
1. `git clone` this repo
2. Open with Unity Hub (Add project from disk)
3. Open scene: (put the scene path here, e.g. `Assets/Scenes/KitchenDemo.unity`)
4. Press Play

## Controls
- Movement: WASD / arrow keys / Xbox controller
- Jump: Space / A on controller

## Current Gameplay
- Collect the fridge crumb -> Roomba activates and chases on line-of-sight.
